# tct-cowork-plugin

This is a Claude Code plugin providing skills for generating and amending Word documents from SAP ticket data.

## Project docs

- [.claude/adding_skill.md](.claude/adding_skill.md) — Process guide for adding a new skill: file structure, SKILL.md conventions, word-mcp-steps/amend-steps patterns, cross-cutting files to update, and pre-implementation review checklist. Read this before designing or implementing any new skill.
